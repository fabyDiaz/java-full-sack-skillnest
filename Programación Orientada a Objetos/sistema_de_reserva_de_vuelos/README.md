# Sistema de Reserva de Vuelos ✈️

## Descripción
Sistema de gestión de reservas para una aerolínea que maneja vuelos domésticos, internacionales y de carga utilizando programación orientada a objetos en Java.

## Concepto Principal: Clases Abstractas

### ¿Qué aprendimos?

**Clases Abstractas** son clases que no se pueden instanciar directamente y sirven como plantilla para otras clases.

**Características clave:**
- Se declaran con la palabra `abstract`
- Pueden tener métodos abstractos (sin implementación) y concretos (con implementación)
- Las clases hijas **deben** implementar todos los métodos abstractos
- Permiten definir comportamiento común y específico
