# Sistema de Gestión de Hospital (Actividad opcional)

## Objetivos
Practicar el diseño de ERD.

Esta tarea implica diseñar un ERD para un sistema de gestión de hospital.  
El sistema debe gestionar información sobre pacientes, médicos, citas, consultorios, enfermeras, visitas, departamentos, historiales médicos, medicamentos y prescripciones.

---

## Requerimientos

- **Pacientes:** ID de paciente, nombre, apellido, fecha de nacimiento, género, dirección.  
- **Médicos:** ID de médico, nombre, apellido, especialidad, número de licencia.  
- **Citas:** ID de cita, ID de paciente, ID de médico, fecha y hora de la cita, estado.  
- **Consultorios:** ID de consultorio, número de consultorio, ubicación.  
- **Enfermeras:** ID de enfermera, nombre, apellido, número de licencia.  
- **Visitas:** ID de visita, ID de paciente, ID de médico, fecha y hora de la visita, diagnóstico.  
- **Departamentos:** ID de departamento, nombre del departamento, ubicación.  
- **Historiales Médicos:** ID de historial, ID de paciente, ID de médico, fecha de creación, detalles médicos.  
- **Medicamentos:** ID de medicamento, nombre del medicamento, dosis, fabricante.  
- **Prescripciones:** ID de prescripción, ID de paciente, ID de médico, ID de medicamento, fecha de prescripción.
