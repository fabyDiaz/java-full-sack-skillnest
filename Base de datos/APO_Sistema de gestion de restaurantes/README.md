# Sistema de Gestión de Restaurante (Actividad opcional)

## Objetivos
Practicar el diseño de ERD.

Esta tarea implica diseñar un ERD para un sistema de gestión de restaurantes.  
El sistema debe gestionar información sobre restaurantes, mesas, clientes, pedidos, menús, cocineros, ingredientes, preparaciones, reservas y facturas.

---

## Requerimientos

- **Restaurantes:** ID de restaurante, nombre del restaurante, dirección, teléfono.  
- **Mesas:** ID de mesa, número de mesa, capacidad, estado de disponibilidad.  
- **Clientes:** ID de cliente, nombre, apellido, teléfono, dirección.  
- **Pedidos:** ID de pedido, ID de cliente, ID de mesa, fecha y hora del pedido, estado.  
- **Menús:** ID de menú, nombre del artículo, descripción, precio.  
- **Cocineros:** ID de cocinero, nombre, apellido, especialidad.  
- **Ingredientes:** ID de ingrediente, nombre del ingrediente, cantidad en stock.  
- **Preparaciones:** ID de preparación, ID de coci
