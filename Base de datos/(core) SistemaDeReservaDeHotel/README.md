# Sistema de Reservas de Hotel (Core)

## Objetivos
Practicar el diseño de ERD.  
Esta tarea implica crear un ERD para un sistema de reservas de hotel. La base de datos debe contener información sobre las habitaciones disponibles, los clientes que realizan las reservas y los detalles de las reservas, incluyendo fechas de llegada y salida.

## Requerimientos

### Habitaciones
- Número de habitación  
- Tipo de habitación  
- Capacidad  
- Precio por noche

### Clientes
- ID de cliente  
- Nombre  
- Apellido  
- Dirección  
- Correo electrónico

### Reservas
- ID de reserva  
- ID de cliente  
- Número de habitación  
- Fecha de llegada  
- Fecha de salida
