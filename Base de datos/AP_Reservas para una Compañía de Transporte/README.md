# Reservas para una Compañía de Transporte (Actividad práctica)

## Objetivos
Practicar el diseño de ERD.

Esta tarea implica diseñar un ERD para un sistema de reservas para una compañía de transporte. El sistema debe gestionar viajes, pasajeros, asientos, reservas, conductores y vehículos.

## Requerimientos

### Viajes
- ID de viaje  
- Origen  
- Destino  
- Fecha de salida  
- Fecha de llegada  
- Precio del boleto

### Pasajeros
- ID de pasajero  
- Nombre  
- Apellido  
- Dirección  
- Correo electrónico

### Asientos
- ID de asiento  
- Número de asiento  
- Estado de disponibilidad

### Reservas
- ID de reserva  
- ID de viaje  
- ID de pasajero  
- ID de asiento  
- Estado de reserva

### Conductores
- ID de conductor  
- Nombre  
- Apellido  
- Número de licencia  
- Vehículo asignado

### Vehículos
- ID de vehículo  
- Número de placa  
- Modelo  
- Capacidad
