# Empresa de Construcción (Actividad práctica)

## Objetivos
Practicar el diseño de ERD.

Esta tarea implica diseñar un ERD para una empresa de construcción que administra proyectos, empleados, clientes, tareas, materiales y equipos. Los empleados están asignados a proyectos y realizan tareas específicas, utilizando materiales y equipos proporcionados por la empresa.

## Requerimientos

### Proyectos
- ID de proyecto  
- Nombre del proyecto  
- Fecha de inicio  
- Fecha de finalización

### Empleados
- ID de empleado  
- Nombre  
- Apellido  
- Puesto  
- Salario

### Clientes
- ID de cliente  
- Nombre del cliente  
- Dirección  
- Correo electrónico

### Tareas
- ID de tarea  
- Descripción de la tarea  
- Fecha de asignación  
- Estado

### Materiales
- ID de material  
- Nombre del material  
- Cantidad en stock  
- Precio por unidad

### Equipos
- ID de equipo  
- Nombre del equipo  
- Estado de disponibilidad
