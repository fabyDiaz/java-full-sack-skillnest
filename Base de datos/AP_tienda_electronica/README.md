# Inventario en Tienda Electrónica (Actividad práctica)

## Objetivos
Practicar el diseño de ERD.

Esta tarea implica diseñar un ERD para un sistema de seguimiento de inventario en una tienda de electrónica. La tienda necesita información sobre los productos disponibles, los proveedores que los suministran y los detalles de los inventarios, incluyendo la cantidad recibida y la fecha de recepción.

## Requerimientos

### Productos
- ID de producto  
- Nombre del producto  
- Marca  
- Precio  
- Cantidad en stock

### Proveedores
- ID de proveedor  
- Nombre del proveedor  
- Dirección  
- Correo electrónico

### Inventarios
- ID de inventario  
- ID de producto  
- ID de proveedor  
- Fecha de recepción  
- Cantidad recibida
