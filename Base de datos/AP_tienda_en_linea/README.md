# Pedidos en Tienda en Línea (Actividad práctica)

## Objetivos
Practicar el diseño de ERD.

Esta tarea implica diseñar un ERD para una tienda en línea que gestiona información sobre clientes, pedidos y productos. Cada pedido está asociado a un cliente y contiene detalles sobre los productos solicitados, como nombre, descripción y precio.

## Requerimientos

### Clientes
- ID de cliente  
- Nombre  
- Apellido  
- Dirección  
- Correo electrónico

### Pedidos
- ID de pedido  
- ID de cliente  
- Fecha de pedido  
- Estado del pedido

### Productos
- ID de producto  
- Nombre del producto  
- Descripción  
- Precio
